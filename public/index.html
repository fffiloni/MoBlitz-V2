Shortcuts

Essentials

Save the current drawing and go for the next blank frame: ctrl+ S
Undo your last pencil move: ctrl+ Z
Play the animation: SPACEBAR
Toggle Onion Skin: O

Tools

Select the pen tool: V
Select the brush tool: B
Select the rough pencil tool: R
Toggle the eraser: E
Select the line tool: L
Change tool size: Up and Down arrows

Timeline & Frames

Display a frame: click on a white rounded rectangle
Go one frame forward: Right arrow
Go one frame backward: Left arrow
Fly hover the timeline: Alt+ mouse hover frames

Actions on frames

Duplicate a frame: ctrl + D
Update a frame: U
Insert a frame: I
Delete a frame: ctrl+E
Throw current drawing and go to blank ahead: X

FAQ

How do i create another layer ?

You can't really create another layer, but you can cheat by using the brush layer or the roughs layer as a secondary plane if you need it.

How do i fix a drawing ?

This feature is in the development pipeline. You can do it though, by pressing the K key. This will fix the current drawing on every frame. But beware: it will not be saved on database, as it is a feature in the making.

How do i duplicate a frame next to it?

To duplicate a frame in the timeline, click on the frame you want to duplicate, then hit ctrl + D on the keyboard.

How do i duplicate a frame ahead?

To duplicate a frame, click on the frame you want to duplicate, then click on the "save & next" button.
This will copy the current displayed drawing on a new frame ahead.
